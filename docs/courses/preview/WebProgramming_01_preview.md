# Web Programming 1° - Crea il tuo primo sito web professionale

## 🚀 Entra nel mondo dello sviluppo Web

Ogni sito Internet che utilizzi ogni giorno, dai social network agli e-commerce, nasce grazie a tre tecnologie fondamentali: **HTML, CSS e JavaScript**.

Con il corso **Web Programming 1°** imparerai a progettare e realizzare pagine web moderne, responsive e interattive, acquisendo le competenze essenziali per iniziare il tuo percorso come sviluppatore Front-End.

Attraverso spiegazioni chiare, esempi pratici ed esercitazioni guidate, scoprirai come trasformare una semplice idea in un sito web funzionante, comprendendo il ruolo di ogni tecnologia e imparando a utilizzarle insieme per creare interfacce professionali.

Che tu voglia avvicinarti alla programmazione, creare un portfolio personale o muovere i primi passi nel mondo dello sviluppo web, questo corso rappresenta il punto di partenza ideale.

---

## Perché seguire questo corso

- 🌐 Impara le tecnologie fondamentali del Web.
- 💻 Realizza pagine web moderne e responsive.
- 🎨 Dai stile ai tuoi siti con CSS.
- ⚡ Aggiungi interattività con JavaScript.
- 🚀 Acquisisci le basi richieste per diventare uno sviluppatore Front-End.

---

## Cosa riuscirai a fare

Al termine del corso sarai in grado di:

- comprendere come funziona il Web e la struttura di una pagina Internet;
- creare pagine HTML ben organizzate;
- progettare layout moderni utilizzando CSS;
- rendere i siti responsive e adattabili a qualsiasi dispositivo;
- aggiungere elementi dinamici e interattivi con JavaScript;
- sviluppare piccoli progetti web completamente funzionanti.

Le competenze acquisite costituiranno la base per affrontare successivamente framework moderni come React, Next.js, Angular o Vue e approfondire lo sviluppo Full Stack.

---

## Cosa imparerai

- Introduzione al Web e al funzionamento di Internet
- Struttura di una pagina HTML
- Tag, elementi e attributi fondamentali
- Formattazione di testi, immagini, tabelle e collegamenti
- Introduzione ai fogli di stile CSS
- Colori, font, layout e responsive design
- Flexbox e principi di impaginazione moderna
- Introduzione a JavaScript
- Variabili, operatori e strutture di controllo
- Funzioni ed eventi
- Manipolazione degli elementi HTML (DOM)
- Creazione di pagine web dinamiche e interattive
- Buone pratiche nello sviluppo Front-End
